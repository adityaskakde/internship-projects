# Crowdfunding Platform Demo

A premium-looking crowdfunding web application built with HTML, CSS, and JavaScript.

## Features

- Create new crowdfunding campaigns
- Set funding goals and deadlines
- Search and sort active campaigns
- Support campaigns with contribution simulation
- Publish and view campaign updates
- Display campaign statistics and progress details
- Browser-local storage persistence

## How to use

1. Open `index.html` in your browser.
2. Create a campaign using the form.
3. Search, sort, and support live campaigns.
4. Open campaign details for updates and funding breakdown.

## Notes

- This is a static frontend application; contributions are simulated in the browser.
- Data is stored locally in `localStorage`.
- The app includes a premium UI style suitable for an internship submission.
